# Frontend Frequência

Repositório responsável pela interface e painel administrativo do Sistema de Coleta de Frequência do Cursinho Popular EACH (CPE). Este sistema fornece as telas para a coleta de presença dos alunos e a visualização de relatórios gerenciais de evasão.

---

## Visão Geral

- **Objetivo:** Oferecer uma interface intuitiva para o registro diário de presença e acompanhamento das métricas de frequência.  
- **Público:** Voluntários do balcão (registro via câmera/manual) e coordenadores (análise do dashboard).  

---

## Tecnologias Utilizadas

- **Framework:** React (Vite)  
- **Linguagem:** TypeScript  
- **Estilização:** Tailwind CSS v3  
- **Roteamento:** React Router Dom
